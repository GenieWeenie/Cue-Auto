# Skill Examples

This folder contains realistic sample skills for authors to copy and adapt.

These examples are intentionally stored under `skills/examples/` so they are **not**
auto-loaded by CueAgent's runtime skill loader.

Included examples:
- `research_brief.py` — structured research brief output with source list normalization
- `release_readiness.py` — release gate checklist scoring and pass/fail summary
- `incident_timeline.py` — incident timeline formatter with severity escalation signal
